github提速

default
2020-02-23 11:20:37 0

由于某些原因，国内访问Github会异常缓慢，在clone仓库时甚至只有10k以下的速度，下载半天有时还会失败需要从头再来，甚是让人恼火。 本文介绍通过修改系统hosts文件的办法，绕过国内dns解析，直接访问GitHub的CDN节点，从而达到加速的目的。不需要科学上网，也不需要海外的服务器辅助。

获取GitHub服务器地址

打开 https://www.ipaddress.com/ 输入域名查询

查询以下三个链接的DNS解析地址

  1. github.com
  2. assets-cdn.github.com
  3. github.global.ssl.fastly.net
  4. api.github.com

打开 host 文件

  1. Mac vim /private/etc/hosts
  2. Windows vi C:\Windows\System32\drivers\etc\hosts 刷新缓存 ipconfig /flushdns
